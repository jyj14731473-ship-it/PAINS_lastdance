# MLB 2021-2025 3범주 분류 베이스라인 실행 기록

## 목적

LAD 단일팀 표본이 작아 MLB 전체팀 5년치 불펜 등판으로 표본을 확장하고, 회귀 대신 3범주 분류 베이스라인을 확정한다.

## 데이터

```text
기간: 2021-03-01 ~ 2025-10-01
대상: MLB 전체팀 Statcast pitch-level 데이터
단위: pitcher-game relief outing
원본 불펜 outing 수: 90,631
```

필터:

```text
투수별 최소 30 IP
투수별 최소 100 BF
```

필터 후:

```text
label 사용 가능 outing 수: 82,809
투수 수: 798
train rows: 74,991
test rows: 7,542
test start: 2025-06-23
```

## 실행 명령

```bash
python compare.py ^
  --input data/outings_mlb_bullpen_2021_2025.parquet ^
  --run-id mlb_bullpen_2021_2025_min30ip100bf_3class_baseline ^
  --output-dir experiments/runs/mlb_bullpen_2021_2025_min30ip100bf_3class_baseline ^
  --min-pitcher-ip 30 ^
  --min-pitcher-bf 100
```

## 베이스라인 모델

```text
classification_residual_tertile_xgboost
```

라벨 기준:

```text
0 = 하/risk
1 = 중/normal
2 = 상/good
```

컷포인트:

```text
train residual 1/3 quantile: -0.02510350989916447
train residual 2/3 quantile:  0.027208668188562973
```

클래스 수:

```text
train: 하 24,997 / 중 24,997 / 상 24,997
test : 하  2,434 / 중  2,713 / 상  2,395
```

## 해석

현재 베이스라인은 무작위 3분류보다 약간 낫다.

```text
무작위 balanced_accuracy 기준선: 약 0.333
현재 balanced_accuracy: 0.366
```

실전적으로는 “정확히 맞히는 모델”이라기보다 `proba_risk`로 위험 후보를 정렬하는 약한 랭킹 모델에 가깝다. `proba_risk` 상위 20%의 실제 risk 비율은 0.390으로, 테스트 전체 risk 비율 0.323 대비 약 1.21배 높다.

## 현재 결론

회귀 모델은 폐기한다. 이후 베이스라인은 이 3범주 분류 모델 하나로 둔다.
